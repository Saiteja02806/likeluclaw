# Web Browser

Use `web_search` and `web_fetch` for current information, research, and fact-checking.

## Rules
- Cite sources with URLs
- Summarize content, don't copy large blocks
- Verify critical claims from multiple sources
- **One search at a time** — wait for each search result before starting the next
- Never run multiple `web_search` calls in parallel or rapid succession
- If you need multiple searches, do them sequentially with a brief pause between each
- Prefer fewer, well-crafted search queries over many narrow ones
